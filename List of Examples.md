The source code is available in this [repo](https://github.com/euonymos/ddn-book-examples):
* `excel` — Excel: B1 = A1 + A2 [[Chapter 2. Taste of Nix]]
* `fib` — Fibonacci numbers in Nix (not Nix the language)
